Division de projet en prototypes

Dans cette étape de réalisation de notre projet nous allons apprendre comment réaliser techniquement notre page web.

Analyse technique

Avant de commencer notre auto-formation, vous devez poser la question : qu’est ce que je doit apprendre pour réaliser notre projet ?

pour répondre à notre question, vous devez analyser techniquement votre projet pour trouver les technique à apprendre et que vous ne maîtriser pas.

Le problème ici, pour les débutants, comme vous, est qu’il ne connais pas encors tous les techniques pour arriver à déterminer les choses à apprendre.

Analyse technique de notre projet

Dans ce projet, permettez-moi de vous proposer une analyse technique adéquate à notre projet, en attenant que vous arriver à vous serez capable de réaliser l’analyse technique toute seul.

Pour réaliser notre projet nous avons besoin de savoir :

  • les notions de base du langage HTML
  • les notions de base du langage CSS
  • le rôle et l’utilisation de la la bibliothèque normalize
  • la disposition en utilisant Float
  • utilisation du Google Fonts
  • Choisir les couleurs pour une page web

Puisque c’est notre premier projet, nous avons bouceau de chose à apprendre. le secret de la réussite est le principe de diviser pour raigné. C’est à dir, diviser le projet en étapes que je les nomes des exemples ou des prototypes.

Je vous propose de réaliser notre projet en trois étapes ou trois prototypes.

Prototype 1

  • les notions de base du langage HTML

Prototype 2

  • les notions de base du langage CSS

Prototype 3

  • structure sémantique d’une page web
  • le rôle et l’utilisation de la la bibliothèque normalize

Prototype 4

  • utilisation du Google Fonts
  • Choisir les couleurs pour une page web

Prototype 5

  • la disposition en utilisant Float

J’ai diviser les technique à apprendre selon la nature de travail et la difficulté.
